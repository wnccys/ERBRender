# <div align="center"> erbr </div>
erbr makes the bound and rendering of a given template inside a given layout allowing parameters to be passed and used inside it;
<br>
## Installation
RubyGems:
```
gem install erbr
```
Gemfile: 
```sh
gem "erbr", '~> 0.0.1'
```
<br>
